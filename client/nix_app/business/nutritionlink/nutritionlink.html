What is NutritionLink™

The NutritionLink™ algorithm checks for the accuracy of nutritional information using a sophisticated set of rules curated by our registered dietitian team.

How NutritionLink™ works

  1. Dietitian Team Maintains Algorithm

    The NutritionLink™ algorithm is powered by a complex set of rules that are regularly updated by our registered dietitian team.

  2. Nutrition Data Imported

    Food brands and database providers upload a list of their existing nutrition data into NutritionLink™ via CSV files, JSON Data, or direct API integration.

  3. Nutrition Data is Validated

    The nutrition values of each food are analyzed using our complex dietitian-curated rule set. If foods do not pass a rule, they are added to the report the the algorithm generates.

  4. Nutrition Data is Validated

    Once a food has passes the algorithm, we update the Nutritionix database to indicate that this food has passed the check. Because the algorithm is regularly updated, foods are automatically checked against the latest version of the algorithm on a regular basis.
